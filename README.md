\# Personal Voice Assistant 🎙️



A Python-based voice assistant that listens to your commands and responds.



\## Features

\- Speech to text using SpeechRecognition

\- Text to speech using pyttsx3

\- Search Google, play YouTube, get time/date

\- Wikipedia search

\- Open websites by voice



\## Installation

pip install -r requirements.txt



\## Run

python assistant.py



\## Commands

| Say | Action |

|-----|--------|

| hello | Greet |

| what time is it | Current time |

| search python | Google search |

| play Believer | YouTube |

| who is Elon Musk | Wikipedia |

| bye | Exit |
